自定义CSS

UI主题自定义面板位于：用户设置 → 主题颜色 上方

使用说明

在自定义CSS中为属性添加标记注释，即可在上方面板中可视化修改图片、文字、颜色和尺寸。

注意：同一面板中标记名称不能重复！
标记格式
图片 /* ggg-img: 名称 */
/* ggg-img-user: 名称 */
/* ggg-img-char: 名称 */
放在 url('...') 的前面（同一行或上一行均可）
文字 /* ggg-text: 名称 */
/* ggg-text-user: 名称 */
/* ggg-text-char: 名称 */
放在 content: "..." 的前面（同一行或上一行均可）
颜色 /* ggg-color: 名称 */
/* ggg-color-user: 名称 */
/* ggg-color-char: 名称 */
放在颜色值或CSS变量声明的前面（同一行或上一行均可）
尺寸 /* ggg-dim: 名称 */
/* ggg-dim-user: 名称 */
/* ggg-dim-char: 名称 */
放在CSS数值属性（如 top、width、height 等）的上一行
使用示例
以下示例美化代码来自呱呱
/* 图片标记 - 标记在上一行 */
#chat .mes .ch_name::before {
    /* ggg-img-char: 闪烁星星图 */
    background-image: url('https://i.postimg.cc/JzNHCLGK/1-1-1.png');
    background-size: auto 100%;
}

/* 图片标记 - 标记在同一行 */
#chat .mes .ch_name::after {
    background-image: /* ggg-img-user: 闪烁星星图 */url('https://i.postimg.cc/mk42wcSs/2-1-1.png');
}

/* 文字标记 */
.mes:not([is_user="true"])::before {
    /* ggg-text: 搜索框文字 */
    content: "crush约我吃日料但我海鲜过敏怎么办T T";
}

/* 颜色标记 - 标记在上一行（推荐） */
:root {
    /* ggg-color: 主题色 */
    --blur-tint-color: rgba(178, 152, 164, 1);
}

/* 颜色标记 - 标记在同一行 */
:root {
    --my-border: /* ggg-color-char: 角色边框色 */ #ff6b9d;
}

/* 尺寸标记 - 用于自定义位置和大小 */
.mes:not([is_user="true"]) .mesAvatarWrapper .mesIDDisplay::after {
    content: "";
    position: absolute;
    /* ggg-dim: 增大下移 */
    top: -45px;
    /* ggg-dim: 增大左移 */
    right: -30px;
    /* ggg-dim: 宽 */
    width: 70px;
    /* ggg-dim: 高 */
    height: 70px;
    /* ggg-img-char: 楼层背图 */
    background: url('https://i.postimg.cc/jqn88QPT/hui.png') no-repeat center;
    background-size: contain;
}

/* 歌词动画中的文字标记 */
@keyframes userLyrics {
    /* ggg-text-user: user歌词1 */
    0%, 14.6% { content: "You and I are a story"; }
    /* ggg-text-user: user歌词2 */
    14.7%, 30.3% { content: "That never gets told"; }
}
自定义 CSS
使用说明

直接写 CSS，保存后立即注入酒馆页面。绑定当前主题，切换主题时自动切换对应的代码。

  • 可用酒馆主题变量：var(--SmartThemeBodyColor)、var(--SmartThemeQuoteColor) 等
  • 加入 /* ggg-color: 名称 */ 标记 → 在「UI主题自定义」面板里可视化拾色修改
  • 覆盖酒馆原样式加 !important

刷新页面自动恢复。与酒馆主题文件无关，可用下方按钮单独导出/导入。

## 代码注入方式
- 「自定义CSS」：直接注入酒馆页面的 <style> 标签，写标准 CSS 即可
- 「HTML注入」：将任意 HTML / JS 插入页面 DOM，<script> 标签会被正确执行

## ggg 特殊标记（写在 CSS 中，让插件面板可视化控制这些值）

★ ggg-color — 颜色拾色器
    .my-el {
        /* ggg-color: 名称 */
        background: #7b7bff;
    }
    /* 或写在同一行属性值前 */
    .my-el { color: /* ggg-color: 名称 */ #fff; }

★ ggg-img   — 图片路径替换
    .avatar { /* ggg-img: 头像 */ background: url('/path/to/img.png'); }

★ ggg-text  — 文字内容替换
    .label::before { /* ggg-text: 标签文字 */ content: "示例"; }

★ ggg-dim   — 数值（px / em / %）调整
    .box { /* ggg-dim: 圆角 */ border-radius: 12px; }

## 常用酒馆主题 CSS 变量（自动跟随当前主题）
    --SmartThemeBodyColor        主文本颜色
    --SmartThemeQuoteColor       引用/强调色（常用于高亮、按钮）
    --SmartThemeBorderColor      边框颜色
    --SmartThemeBlurTintColor    UI 半透明背景
    --SmartThemeChatTintColor    聊天区背景
    --SmartThemeShadowColor      阴影色

## HTML 注入注意事项
- 覆盖酒馆原有样式需加 !important
- 悬浮元素用 position:fixed（不填目标选择器时插到 <body> 末尾）
- 颜色优先用 CSS 变量或 ggg-color 标记，避免硬编码
- JS 脚本直接写 <script>...</script> 放入注入框，插件会自动执行

使用说明

将任意 HTML 代码（包括 <script> 脚本）插入到酒馆页面 DOM 中，绑定当前主题。

三个字段的含义：

  • 片段名称：只是给这条记录起个你自己看得懂的名字，显示在下方列表里，不影响任何功能。
  • 目标选择器：留空则插到 <body> 末尾（最常用）；也可以填酒馆某个元素的 CSS 选择器（如 #chat、.mes_block）把代码插到指定位置。
  • 插入位置：相对目标元素，可选「内部末尾 / 内部开头 / 元素之后 / 元素之前」。不填目标选择器时此项无效。

常见用途：

  • 悬浮装饰元素 → 不填选择器，代码里用 position:fixed 定位
  • 运行 JS 脚本 → 把 <script>...</script> 直接粘进来，插件会正确执行它
  • 插入到聊天区前 → 目标选择器填 #chat，位置选「内部开头」

刷新页面自动重新注入，主题切换时随之切换。
