美化 - UI主题自定义

在自定义CSS中添加标记注释，即可在面板中可视化修改图片、文字和颜色。

UI主题自定义面板位于 用户设置 → 主题颜色 上方。

注意：标记注释中的名称不能重复！不同标记项请使用不同的名称。

标记格式
图片 /* ggg-img: 名称 */
/* ggg-img-user: 名称 */
/* ggg-img-char: 名称 */
放在 url('...') 的前面（同一行或上一行均可）
文字 /* ggg-text: 名称 */
/* ggg-text-user: 名称 */
/* ggg-text-char: 名称 */
放在 content: "..." 的前面（同一行或上一行均可）
颜色 /* ggg-color: 名称 */
/* ggg-color-user: 名称 */
/* ggg-color-char: 名称 */
放在颜色值或CSS变量声明的前面（同一行或上一行均可）
SVG文字 /* ggg-svg-text: 名称 */
/* ggg-svg-text-user: 名称 */
/* ggg-svg-text-char: 名称 */
SVG颜色 /* ggg-svg-color: 名称 */
/* ggg-svg-color-user: 名称 */
/* ggg-svg-color-char: 名称 */
用于 SVG data URL 中的文字和颜色。放在含有 url("data:image/svg+xml,...") 的属性前面
使用示例
以下示例美化代码来自呱呱
/* 图片标记 - 标记在上一行 */
#chat .mes .ch_name::before {
    /* ggg-img-char: 闪烁星星图 */
    background-image: url('https://i.postimg.cc/JzNHCLGK/1-1-1.png');
    background-size: auto 100%;
}

/* 图片标记 - 标记在同一行 */
#chat .mes .ch_name::after {
    background-image: /* ggg-img-user: 闪烁星星图*/url('https://i.postimg.cc/mk42wcSs/2-1-1.png');
}

/* 文字标记 */
.mes:not([is_user="true"])::before {
    /* ggg-text: 搜索框文字 */
    content: "crush约我吃日料但我海鲜过敏怎么办T T";
}

/* 颜色标记 - 标记在上一行（推荐） */
:root {
    /* ggg-color: 主题色 */
    --blur-tint-color: rgba(178, 152, 164, 1);
    /* ggg-color: 消息背景色 */
    --color-light-grey-1: rgb(85, 85, 85);
}

/* 颜色标记 - 标记在同一行 */
:root {
    --my-border: /* ggg-color-char: 角色边框色 */ #ff6b9d;
}

/* SVG文字和颜色标记 - 用于SVG data URL */
#send_textarea:placeholder-shown {
    /* ggg-svg-color: 输入框文字颜色 */
    /* ggg-svg-text: 输入框文字 */
    background-image: url("data:image/svg+xml,...") !important;
}

/* 歌词动画中的文字标记 */
@keyframes userLyrics {
    /* ggg-text-user: user歌词1 */
    0%, 14.6% { content: "You and I are a story"; }
    /* ggg-text-user: user歌词2 */
    14.7%, 30.3% { content: "That never gets told"; }
}
